# vor(Validator)

```javascript
vor.typeOf(target).is(a=>a.primitive.number.integer.positive)
vor.typeOf(target).expect(a=>a.primitive.number.integer.positive)
vor.typeOf(target).is(a=>a.reference.instance)
vor.typeOf(target).is(a=>a.reference.instance.of(SomeClass))
vor.valueOf(target).is(...candidates)
vor.valueOf(target).expect(...candidates)
vor.rangeOf(target).is(min, max)
vor.rangeOf(target).expect(min, max)
vor.self(owner).has('name', a=>a.descriptor.a.g);
vor.proto(owner).has('name', a=>a.descriptor.a.g);
vor.chain(owner).has('name', a=>a.descriptor.a.g);
```
```javascript
// 対象の型はプリミティブ型の正数である
// type of target is (a.primitive.number.integer.positive)
vor.typeOf(target).is(a=>a.primitive.number.integer.positive)
// 対象の型はプリミティブ型の正数であることを期待する。。
// type of target expect (a.primitive.number.integer.positive)
vor.typeOf(target).expect(a=>a.primitive.number.integer.positive)
// 対象自身は'name'という名前のゲッターを持っている。
// The target itself has a getter named 'name'.
vor.itself(target).has('name', a=>a.descriptor.a.g)
// 対象のプロトタイプは'name'という名前のゲッターを持っている。
// The target prototype has a getter named 'name'.
vor.protoOf(target).has('name', a=>a.descriptor.a.g)
vor.protoOf(target).has('name', a=>a.descriptor.a.g)
// 鎖(対象自身やそのプロトタイプチェーンにあるいずれか)は'name'という名前のゲッターを持っている。
// vor.
```
対象の系譜は'name'という名前のゲッターを持っている。
The target lineage has a getter named 'name'.

対象の先祖は'name'という名前のゲッターを持っている。
The target ancestor has a getter named 'name'.

* 対象自身
* 対象の一つ上のプロトタイプ
* 対象より一つ以上上のプロトタイプ
* 対象を含む一つ以上上のプロトタイプ

対象自身と、対象を含む一つ以上上のプロトタイプの二つがあれば良いはずだ。
